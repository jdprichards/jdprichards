NeuroMap Exporter

NeuroMap Exporter is a program designed to help users convert text files (.txt, .csv, .tsv) from NeuroMap Exporter (Delsys Inc., MA, USA) to Visual 3D (HAS Motion, Kingston, CA). Using different methods, there are currently three features that have been implemented.

NeuroMap Exporter:

Convert text based files into a format that can be used by Visual3D

EMG-IMU Upsampler:

Upsample and conver text files which contain data with different sample rates
